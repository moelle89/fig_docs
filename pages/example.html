Button groups are containers for related buttons. They help organize buttons with similar actions or functions, making interfaces cleaner and more intuitive for users.


Button Group Examples

Horizontal Button Group

Horizontal Layout

Horizontal button groups arrange buttons side by side, creating a unified control that clearly shows related options. This layout is ideal for toolbar actions, pagination controls, or any situation where space allows for a horizontal arrangement.

In our implementation, buttons share borders to visually connect them while maintaining clear separation between options.

Vertical Button Group

Vertical Layout

Vertical button groups stack buttons on top of each other, which is useful for mobile interfaces or when horizontal space is limited. This arrangement works well in side navigation, dropdown menus, or option lists.

Our vertical button groups maintain consistent spacing and visual connection between buttons, with options for various sizes and styles.
