How to Use

1. Include the Required Files

Add the following to your HTML:

<link rel="stylesheet" href="css/prop-table.css">
<script src="js/prop-table-generator.js"></script>

2. Create a Table from JSON Data

Use the createPropTable function:

// Example data
const buttonData = {
    'button label': 'Click me',
    'icon after': true,
    'icon before': false,
    'size': 'md',
    'type': 'Primary',
    'state': 'Default',
    'corner radius': 'Default'
};

// Create the table
createPropTable(buttonData, 'Button Properties', 'container-id');

3. Create a Table from TypeScript Interface

Use the createPropTableFromInterface function:

const interfaceString = `
interface Button {
    button label: string; 
    icon after swap: ReactNode; 
    icon before swap: ReactNode; 
    icon after: boolean; 
    icon before: boolean; 
    size: 'sm' | 'md' | 'lg' | 'xl' | 'xxl'; 
    type: 'Primary' | 'Secondary' | 'Tertiary' | 'Link 01' | 'Link 02'; 
    state: 'Default' | 'Hover' | 'Click' | 'Focus' | 'Disabled'; 
    corner radius: 'Default' | 'Pill'; 
}`;

// Create the table
createPropTableFromInterface(interfaceString, 'Button Interface', 'container-id');
